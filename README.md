#project Title 
    Task Management 

#Short Descrbition 

 This is a Full Stack Task Management System (MERN Stack) built using React, Node.js, Express, and MongoDB. It allows users to create, view, update, delete, and search tasks efficiently. Each task includes a title, description, status (Pending or Completed), and due date. The system uses a REST API for communication between frontend and backend, and MongoDB is used to store and manage task data. It is designed as a beginner-to-intermediate level portfolio project to demonstrate CRUD operations, API integration, and modern web development skills.

 #Features
 Create Tasks – Add new tasks with title, description, and due date
 View Tasks – Display all tasks in a clean table format
 Edit Tasks – Update task details like title, description, and due date
 Complete Tasks – Change task status from Pending to Completed
 Delete Tasks – Remove unwanted tasks permanently
 Search Tasks – Quickly find tasks by title using search
 Full Stack Integration – React frontend connected to Node.js + Express backend
 Database Storage – MongoDB used for persistent task management
 REST API – Clean API structure for all CRUD operations
